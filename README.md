(aka:ask AI MJPEG stream)
基于现有人工智能的MJPEG图像识别工具
仅限windows平台
>[!notice] 本软件本作为闭源自用小工具，现在脱敏修改后发布到github上备份，本readme部分内容与实际使用不相符

### 软件功能
电脑手机都安装对应的软件之后可以在电脑上捕获手机屏幕内容（普通应用权限，不适用于隐私界面）并通过AI解释屏幕内容，最初被用来刷网课

### >使用方法<
在手机上安装文件夹内的screenstream(也可在f-droid上下载)
将电脑和手机连至同一局域网络
打开手机上的screenstream软件
将模式设置为本地模式（MJPEG）
禁用密码（PIN码）
点击“开始流”
电脑端打开软件launch.exe
>软件现在不发布发行版，你可以修改代码后手动生成二进制打包程序launch.exe或者执行python src/main.py

点击左侧默认值按钮
将文本框中的IP部分替换为手机上展示的IP
点击右侧获取图像按钮
左侧将展示手机屏幕当前的画面
可以点击旋转按钮进行旋转
点击上传图像可以让人工智能进行画面分析
获取并上传是前两个按钮的结合
上传图像后右侧识别结果区域将展示来自人工智能的答复

### 项目初始化
这是一个简单的项目初始化方案，适用于github版本
1. 创建venv虚拟环境
2. 安装requirements.txt里面的模块，务必保证flet模块的版本不变
3. 修改src/doubaoAI.py 在里面填写上自己的火山引擎API_KEY并换上仍在支持的视觉模型(旧的模型会逐步下架，需要定期更新)
4. [可选]修改src/doubaoAI.py里aiAsk()函数里question的默认参数为你想要的提示词
5. 运行main.py文件

简易打包流程
1. 安装pyinstaller包(pip install <包名>)
2. 参考打包命令 pyinstaller --clean --noconfirm main.spec
3. 访问dist目录查看带有运行时的exe程序，发布时要发布真个dist
